# share

Helper mínimo de Web Share API hospedado em GitHub Pages.

Demonstração:

- https://jrmsrs.github.io/share?title=Meu%20T%C3%ADtulo&text=Mensagem%20legal&url=https%3A%2F%2Fexemplo.com

Use os parâmetros `title`, `text` e `url` na query string. A página aguarda um gesto do usuário (clique ou tecla) para acionar `navigator.share()` conforme exigido pelos navegadores modernos.
